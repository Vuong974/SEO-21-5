# Workstreams

Thu muc nay dung de quan ly nhieu nhom cong viec trong cung workspace ma khong tron lan voi nhau.

## Nguyen tac

- Moi cong viec moi nen thuoc mot workstream ro rang.
- Moi dau viec lon nen co thu muc theo ngay hoac ten chien dich, vi du `2026-05-27-bao-cao-tong-hop/`.
- Moi dau viec nen copy mau tu `task-template.md` de ghi muc tieu, input, output va trang thai.
- Khong di chuyen file cu neu script dang phu thuoc vao duong dan do. Thay vao do, ghi chu file lien quan trong README cua workstream.
- File dung chung cho nhieu workstream dat trong `shared/`.

## Cac workstream

- `seo/`: cong viec SEO, tu khoa, GSC, WordPress SEO, internal linking.
- `general-reports/`: bao cao tong hop, KPI, dashboard, file Excel/CSV/Markdown bao cao chung.
- `content/`: bai viet, outline, content standards, brand framework.
- `automation/`: workflow tu dong hoa, Apps Script, import/export Google Sheet, WordPress posting.
- `shared/`: template, quy uoc, checklist va tai nguyen dung chung.

## Cach bat dau mot viec moi

1. Chon workstream dung muc dich.
2. Tao thu muc theo mau `YYYY-MM-DD-ten-viec/`.
3. Copy noi dung `task-template.md` vao `README.md` trong thu muc do.
4. Dat file nguon vao `source/`, file ket qua vao `output/`, va ghi lai cach tao ket qua.
