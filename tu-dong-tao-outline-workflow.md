# Workflow thu cong trong Codex: tao outline SEO bang Google Docs

## Muc tieu

Khi ban nhap `Tu khoa chinh` va `Tu khoa phu` vao Google Sheet, Codex se tao mot Google Docs rieng chua outline bai viet SEO, mo full quyen truy cap bang link cho Google Docs neu connector hien tai co cong cu cap quyen, roi ghi link Google Docs vao cot `Outline`.

Workflow nay chay thu cong trong Codex. No khong tu dong chay nen.

## Cau truc cot

| Cot | Ten cot | Vai tro |
| --- | --- | --- |
| A | Tu khoa chinh | Keyword chinh cua bai viet |
| B | Tu khoa phu | Keyword phu, semantic keyword, bien the can chen them |
| C | Outline | Link Google Docs chua outline SEO |

## Cach nhan dien dong can xu ly

Codex se xu ly cac dong thoa dieu kien:

1. Cot `Tu khoa chinh` co du lieu.
2. Cot `Outline` dang trong hoac dang chua outline text cu can chuyen sang Google Docs.

Neu cot `Outline` da co link Google Docs, Codex se bo qua dong do, tru khi ban yeu cau tao lai.

## Quyen chia se Google Docs

Sau khi tao Google Docs, Codex can thuc hien theo thu tu:

1. Tao Google Docs va ghi noi dung outline.
2. Mo quyen truy cap link cho Google Docs truoc khi cap nhat vao sheet.
3. Muc quyen bat buoc uu tien: `Anyone with the link - Editor` de nguoi co link co full quyen chinh sua.
4. Neu connector hoac chinh sach Drive khong cho phep dat `Editor` cong khai, dat muc cao nhat co the thuc hien duoc, vi du `Anyone with the link - Viewer` hoac `Commenter`, va bao lai ro ly do.
5. Gan link Google Docs vao dung o cot `Outline` cua dong dang xu ly sau khi da xu ly quyen truy cap.
6. Doc lai cot `Outline` va metadata quyen chia se cua Google Docs de xac nhan link va trang thai cap quyen.

## Quy tac viet outline

- H1 phai chua tu khoa chinh hoac bien the gan nhat.
- Meta title nen duoi 60 ky tu neu co the.
- Meta description nen duoi 155 ky tu neu co the.
- H2/H3 phai bao phu y dinh tim kiem, khong nhoi keyword.
- Tu khoa phu phai duoc phan bo tu nhien vao cac H2/H3 hoac ghi chu SEO.
- Neu tu khoa phu khac chu de voi tu khoa chinh, chen muc so sanh hoac phan biet thay vi ep cung intent.
- Them FAQ de bat long-tail va People Also Ask.
- Them CTA neu keyword co kha nang chuyen doi.

## Quy trinh Codex se thuc hien

1. Mo Google Sheet bang connector.
2. Doc metadata va header de xac dinh dung cot.
3. Tim dong can tao/chuyen outline.
4. Tao Google Docs voi ten file: `Outline SEO - [Tu khoa chinh]`.
5. Ghi noi dung outline vao Google Docs.
6. Mo full quyen truy cap link cho Google Docs theo muc `Anyone with the link - Editor`.
7. Neu khong mo duoc quyen `Editor`, dung muc quyen cao nhat connector/chinh sach Drive cho phep va ghi ro trong bao cao.
8. Ghi link Google Docs vao cot `Outline`.
9. Kiem tra lai cac dong da ghi va metadata quyen chia se cua Google Docs.
10. Bao cao so Google Docs da tao, link da cap nhat, va trang thai cap quyen.
