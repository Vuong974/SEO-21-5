# Reports

Thu muc nay dang chua bao cao lich su va output tu cac script cu.

Voi cong viec moi:

- Bao cao SEO dat trong `workstreams/seo/`, hoac tiep tuc xuat vao `reports/` neu script cu dang mac dinh duong dan nay.
- Bao cao chung dat trong `workstreams/general-reports/`.
- Backup WordPress va output tu script cu co the giu trong `reports/` de dam bao truy vet.

Khi can don dep, chi di chuyen file sau khi da kiem tra script nao dang doc/ghi duong dan do.
